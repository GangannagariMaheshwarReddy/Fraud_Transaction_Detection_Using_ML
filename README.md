# Fraud Detection Project

A Machine Learning-based Fraud Detection System built using Python, Scikit-learn, and Streamlit.  
This project predicts whether a financial transaction is fraudulent based on transaction details.

---

## 🚀 Features

- Fraud transaction prediction
- Streamlit web application
- Machine Learning pipeline
- User-friendly interface
- Real-time prediction
- Trained model saved using Joblib
